# Lab 1: Understanding ORM with a Retail Inventory System

## Scenario

Build an inventory management system for a retail store that tracks products, categories, and stock levels in SQL Server.

## Implemented

- Explains ORM mapping between C# classes and relational tables.
- Lists EF Core productivity, maintainability, and SQL abstraction benefits.
- Compares EF Core with EF Framework.
- Notes EF Core 8.0 features: JSON columns, compiled models, interceptors, and bulk operation improvements.
- Includes the console project and EF Core package references through the module-level build file.

## Commands

```powershell
dotnet run --project .\Lab01-Understanding-ORM.csproj
```
