# Lab 4: Inserting Initial Data into the Database

## Scenario

The store manager adds initial product categories and products.

## Implemented

- Uses `AddRangeAsync` for `Electronics` and `Groceries`.
- Uses `AddRangeAsync` for `Laptop` and `Rice Bag`.
- Persists with `SaveChangesAsync`.

## Commands

```powershell
dotnet run --project .\Lab04-Insert-Initial-Data.csproj
```
